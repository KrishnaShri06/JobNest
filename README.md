## рџљЂ Setup Instructions

### Prerequisites

1. **Java Development Kit (JDK) 21**
   - Download from [Oracle](https://www.oracle.com/java/technologies/downloads/)
   - Verify: `java -version`

2. **Apache Tomcat 10.1**
   - Download from [Apache Tomcat](https://tomcat.apache.org/download-10.cgi)
   - Extract to a directory (e.g., `C:\Program Files\Apache Software Foundation\Tomcat 10.1`)

3. **MySQL Server**
   - Ensure MySQL is installed and running
   - Default port: 3306

4. **MySQL Connector/J**
   - Download from [MySQL Downloads](https://dev.mysql.com/downloads/connector/j/)
   - Extract the JAR file for later use

### Database Setup

1. Open MySQL Command Line or MySQL Workbench
2. Run the SQL script:
   ```bash
   mysql -u root -p < sql/init.sql
   ```
   Or manually execute the contents of `sql/init.sql`

3. Update database credentials in `src/db.properties`:
   ```properties
   db.username=root
   db.password=your_password
   ```

### Build & Deployment

#### Option 1: Using Build Script (Recommended)

1. **Update Tomcat path** in `build.bat`:
   ```batch
   set TOMCAT_LIB=C:\Program Files\Apache Software Foundation\Tomcat 10.1\lib
   ```

2. **Run the build script**:
   ```bash
   build.bat
   ```

3. **Copy MySQL Connector** to `build\WEB-INF\lib\` when prompted

4. **Deploy the WAR file**:
   - Copy `dist\JobApp.war` to Tomcat's `webapps` folder
   - Tomcat will auto-deploy the application

#### Option 2: Manual Compilation

1. **Compile Java files**:
   ```bash
   javac -d build/WEB-INF/classes -cp "C:\Program Files\Apache Software Foundation\Tomcat 10.1\lib\*" src/com/jobnest/*/*.java
   ```

2. **Copy resources**:
   ```bash
   copy src\db.properties build\WEB-INF\classes\
   xcopy web\* build\ /E /I /Y
   ```

3. **Add MySQL Connector**:
   - Copy `mysql-connector-java-x.x.x.jar` to `build\WEB-INF\lib\`

4. **Create WAR file**:
   ```bash
   cd build
   jar -cvf ..\JobApp.war *
   ```

5. **Deploy** to Tomcat's `webapps` folder

### Running the Application

1. **Start Tomcat**:
   - Windows: Run `bin\startup.bat` in Tomcat directory
   - Linux/Mac: Run `bin/startup.sh`

2. **Access the application**:
   ```
   http://localhost:8080/JobApp
   ```

3. **Test the workflow**:
   - View job listings on home page
   - Click "Apply Now" on any job
   - Fill out the application form
   - Submit and verify success message
   - Check MySQL database for stored application

## рџ—„пёЏ Database Schema

### Jobs Table
```sql
CREATE TABLE jobs (
    id INT PRIMARY KEY AUTO_INCREMENT,
    title VARCHAR(200) NOT NULL,
    company VARCHAR(150) NOT NULL,
    description TEXT NOT NULL,
    location VARCHAR(100) NOT NULL,
    salary VARCHAR(50),
    posted_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Applications Table
```sql
CREATE TABLE applications (
    id INT PRIMARY KEY AUTO_INCREMENT,
    job_id INT NOT NULL,
    applicant_name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    resume_text TEXT,
    applied_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (job_id) REFERENCES jobs(id)
);
```

## рџђ› Troubleshooting

### Common Issues

**1. ClassNotFoundException: com.mysql.cj.jdbc.Driver**
- Solution: Ensure MySQL Connector JAR is in `WEB-INF/lib/`

**2. Cannot connect to database**
- Check MySQL is running
- Verify credentials in `db.properties`
- Check database name is `jobnest`

**3. 404 Error**
- Ensure WAR file is deployed to Tomcat's `webapps` folder
- Check Tomcat is running
- Verify URL: `http://localhost:8080/JobApp`

**4. Compilation errors**
- Ensure Java 21 is installed
- Update TOMCAT_LIB path in build.bat
- Check all dependencies are present

**5. Servlets not loading**
- Verify `web.xml` is in `WEB-INF/` folder
- Check servlet class names match in web.xml
- Review Tomcat logs in `logs/catalina.out`
